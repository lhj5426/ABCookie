# Cookie Profile Switcher 图标制作说明

## 需要的图标尺寸

Chrome扩展程序需要以下4个尺寸的图标：

- **icon16.png** - 16x16像素 (扩展程序管理页面小图标)
- **icon32.png** - 32x32像素 (Windows等系统的标准图标)
- **icon48.png** - 48x48像素 (扩展程序管理页面)
- **icon128.png** - 128x128像素 (Chrome应用商店和安装对话框)

## 设计建议

### 主题元素
- **Cookie图标** 🍪：体现核心功能
- **切换箭头**：表示配置文件切换功能
- **简洁设计**：在小尺寸下依然清晰可见

### 配色方案
- **主色调**：#667eea (渐变蓝紫色，与界面保持一致)
- **辅助色**：#764ba2 (深紫色)
- **背景色**：白色或透明

### 设计要求
1. **简洁明了**：图标应该易于识别和理解
2. **可缩放性**：在不同尺寸下都要清晰
3. **品牌一致性**：与应用界面风格保持一致
4. **平台适配**：符合Chrome扩展程序设计规范

## 制作方法

### 方法一：使用设计软件
1. **Adobe Illustrator / Photoshop**
   - 创建128x128的主图标
   - 向下缩放生成其他尺寸
   - 导出为PNG格式

2. **Figma / Sketch**
   - 在线设计工具，方便协作
   - 支持多尺寸导出

3. **GIMP (免费)**
   - 开源图像编辑软件
   - 功能强大，完全免费

### 方法二：使用在线工具
1. **Canva**
   - 模板丰富，操作简单
   - 直接输出PNG格式

2. **IconPark** (字节跳动)
   - 丰富的图标库
   - 可自定义颜色和样式

3. **Flaticon**
   - 大量免费图标
   - 可编辑和自定义

## 快速制作步骤

### 基础图标制作
1. **选择一个Cookie表情符号** 🍪
2. **添加切换箭头**（可选）
3. **应用渐变色彩**
4. **调整为合适的尺寸**

### 批量生成不同尺寸
```bash
# 如果你有ImageMagick工具，可以用命令行批量转换
convert icon128.png -resize 48x48 icon48.png
convert icon128.png -resize 32x32 icon32.png  
convert icon128.png -resize 16x16 icon16.png
```

## 临时解决方案

如果暂时没有图标，可以：

1. **使用文字图标**：创建包含"CP"字母的简单图标
2. **使用表情符号**：直接截图🍪表情符号并调整尺寸
3. **使用在线生成器**：快速生成基础图标

## 文件命名和格式

- 文件格式：**PNG**
- 文件名：**严格按照要求命名**
  - icon16.png
  - icon32.png
  - icon48.png
  - icon128.png
- 背景：**透明背景**（推荐）
- 质量：**最高质量**，避免压缩失真

## 注意事项

1. **版权问题**：确保使用的图标素材没有版权限制
2. **文件大小**：控制在合理范围内，避免过大
3. **清晰度**：特别注意16x16尺寸的清晰度
4. **一致性**：所有尺寸的图标应保持视觉一致性

完成图标制作后，将4个PNG文件放置在此icons目录中即可。
