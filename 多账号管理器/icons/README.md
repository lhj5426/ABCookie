# 图标设计规范

## 设计理念
新版图标采用现代简约风格，结合了“多账号管理”与“安全隐私”的核心概念。
- **视觉隐喻**：
  - **层叠卡片**：象征多账号切换与管理功能。
  - **用户头像**：明确指代“账号/身份”。
  - **盾牌轮廓（隐含）**：通过卡片的稳固堆叠传达安全感。

## 色彩规范
图标采用清新的蓝色系渐变，与产品主色调保持一致，传达专业、科技与信任感。

| 颜色名称 | Hex 代码 | 使用场景 |
| :--- | :--- | :--- |
| **Primary Blue** (主色) | `#2563eb` - `#60a5fa` | 前景主卡片渐变，用于吸引视觉焦点。 |
| **Secondary Blue** (辅色) | `#bfdbfe` - `#dbeafe` | 背景副卡片渐变，用于营造层次感。 |
| **White** (白色) | `#ffffff` | 用户头像与图标主体，保证高对比度与清晰度。 |
| **Shadow** (阴影) | `#1e3a8a` (25% Opacity) | 卡片投影，增加立体感与悬浮感。 |

## 尺寸与用途
图标提供多种尺寸以适应不同的显示环境，确保在任何分辨率下清晰可辨。

| 尺寸 (px) | 格式 | 用途 |
| :--- | :--- | :--- |
| **16x16** | PNG | 浏览器工具栏图标、标签页图标 (Favicon)。设计上进行了简化，确保小尺寸下的识别度。 |
| **32x32** | PNG | Windows 任务栏、高分屏工具栏图标。 |
| **48x48** | PNG | 扩展管理页面图标。 |
| **128x128** | PNG | Chrome 网上应用店图标、安装时的高清预览图标。 |
| **Vector** | SVG | 原始矢量设计稿，用于生成任意尺寸的导出文件或营销物料。

## 文件结构
- `icon.svg`: 矢量源文件
- `icon-16.png`: 16px 导出文件
- `icon-32.png`: 32px 导出文件
- `icon-48.png`: 48px 导出文件
- `icon-128.png`: 128px 导出文件
